# Complete Example

This example demonstrates how to use the `alicloud-private-link-vpc-endpoint` module to create a complete PrivateLink setup with both the VPC endpoint and its associated service.

## Usage

To run this example, execute the following commands:

```bash
terraform init
terraform plan
terraform apply
```

After you're done, clean up the resources:

```bash
terraform destroy
```

## Cost

**Note:** This example will create resources that cost money. Run `terraform destroy` when you don't need them anymore.

## Requirements

No special requirements beyond having an Alibaba Cloud account with appropriate permissions.
